문서객체모델

CSS에서는 대상에 접근하기 위해서 선택자를 사용했지만 javascript에서는 문서객체모델을 사용합니다.

노드란 문서 상의 객체를 구분짓는 하나하나의 덩어리를 말하며 문서객체모델에는 여러 노드가 있습니다.

  • 요소노드 element node
  • 속성노드 attribute node
  • 텍스트노드 text node

노드는 새로운 것이 아닙니다.

  • 지금까지 다루었던 태그들(h1, li, p)이 요소노드입니다.
  • html lang="ko"에서의 lang 속성이 속성노드입니다.
  • 요소노드가 감싸고 있는 컨텐츠가 텍스트노드입니다.
